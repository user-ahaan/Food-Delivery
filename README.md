# Food Delivery Application

Food delivery app with React frontend, admin panel, and Node.js backend.

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express + MongoDB
- **Payment**: Stripe

### Installation

# Clone repository
git clone https://github.com/user-ahaan/Food-Delivery.git
<br>
cd Food-Delivery

# Backend setup
cd backend
npm install

# Frontend setup
cd ../frontend
npm install

# Admin setup
cd ../admin
npm install

## Run Application
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend
cd frontend && npm run dev

# Terminal 3 - Admin
cd admin && npm run dev
